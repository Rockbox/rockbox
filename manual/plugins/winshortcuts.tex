\subsection{Windows Shortcuts}
\label{ref:Winshortcutsplugin}

This plugin allows to use shortcuts that are created with the Microsoft
Windows Explorer. In rockbox, this shortcuts will show up as \fname{.lnk}
files. To use a shortcut, just ``play'' the \fname{.lnk} file from the file
browser. This will leave you within the linked directory, or with the linked
file selected in the file browser. You can then play the file (or do whatever
you want); it will not be ``played'' automatically.
Only relative links across the same volume are supported.

\note{You may like to use native rockbox shortcuts instead. These can be
    created from within rockbox and have advanced capabilities.
    See \reference{ref:Shortcutsplugin} for more information.}
